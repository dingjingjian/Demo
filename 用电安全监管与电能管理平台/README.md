#智慧式用电安全监管与电能管理平台
主要结合了 layui 与 echart

index.html        管理平台首页
details.html      查看报警详情
page.html         页面访问详情
warn.html         全部报警页面
leakage.html      漏电流页面
temperature.html  温度页面
OUvoltage.html    过欠压页面
overload.html     过流过载页面
leakageW.html     漏电报警页面
temperatureW.html 温度报警页面
sparkingW.html    打火报警页面
shortW.html       短路报警页面
surgeW.html       浪涌报警页面
power.html        电量页面
power2.html       每户的用电情况
load.html         负载页面
control.html      控制管理页面
